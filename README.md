# lawschool.fit

The ultimate law school decision tool. Compare scholarship offers, BigLaw outcomes, and total cost of attendance with real-time ROI calculations and debt payoff simulations.

I built this tool for my own law school decision-making process because I found it difficult to compare disparate scholarship offers against long-term career outcomes. I hope it helps others make more informed choices.

## How to Navigate the Code

- `/src/App.tsx`: The core React application, containing the main UI, dashboard, arena (versus mode), and school editor.
- `/src/data.ts`: Contains the core logic for ROI calculations, data types, and export functionality (CSV/Excel).
- `/src/utils/sounds.ts`: Handles the audio feedback for user interactions.
- `/metadata.json`: Application metadata.
- `/index.html`: Main entry point with SEO and OpenGraph tags.
